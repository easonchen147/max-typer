# Max Typer

<div align="center">

### 把“我不想练打字”慢慢练成“再来一局”

面向儿童与青少年的英文键盘训练游戏。  
把基础练习、街机玩法、数据复盘和成长激励，串成一条顺手的学习闭环。

`React 19` `TypeScript` `Vite` `Zustand` `Dexie` `Phaser 3` `Vitest`

</div>

---

## 这是什么

`Max Typer` 是一个网页端英文打字训练项目，核心思路很简单：

不是让孩子和键盘硬碰硬，而是把“练习”包装成“能玩、能看见进步、还能继续下一局”的过程。

它把基础键位训练、字母反应训练、单词持续输出训练和数据复盘做成一个闭环。你可以先练，再玩，再看数据，再补弱项。键盘不会自己变熟，幸好这个项目会帮你记住哪里还不熟。

## 为什么做它

很多打字练习产品有两个极端：

1. 太像作业，孩子一看就想先喝水、再发呆、最后消失。
2. 太像小游戏，热闹有了，但练完以后并不知道到底哪里进步了。

`Max Typer` 想走中间那条更靠谱的路：

- 让训练足够像游戏，先把“愿意开始”这件事解决。
- 让反馈足够像学习工具，保证每一次错误都能留下价值。
- 让成长足够可视化，让孩子、家长、老师都能看懂进度。

一句话总结：把“我得练打字”这件事，尽量练成“我再来一把”。

## 功能全览

### 1. 训练模式

| 模式 | 主要玩法 | 训练重点 | 当前实现亮点 |
| --- | --- | --- | --- |
| 基础练习 | 根据提示输入字母、单词或短语 | 键位记忆、正确率、输入节奏 | 虚拟键盘高亮、指法提示、即时正误反馈、弱项加权出题 |
| 水果忍者 | 按下对应字母切开水果，躲开炸弹 | 字母反应、连击稳定度、节奏感 | 连击得分、炸弹惩罚、难度节奏差异、街机式反馈 |
| 太空侵略者 | 输入单词击落敌机，顶住波次推进 | 单词识别、持续输入、专注与耐力 | 波次节奏控制、Boss 前过渡提示、单词锁定输入、词库联动 |
| 数据中心 | 训练后查看个人表现与推荐补练项 | 复盘、补弱、长期成长 | 键位热力图、趋势图、弱项快捷强化、难度解锁进度 |

### 2. 学习辅助系统

- 支持首次进入引导，新用户不会一上来就被键盘教育。
- 支持“开始推荐训练”，系统会根据当前进度推荐最适合的模式。
- 支持“今日优先键位”，根据近期弱项和日期动态生成每日重点。
- 支持“每日小目标”，让训练不是盲打，而是有当天节奏。
- 支持“成就系统”，例如连续训练、体验双街机模式、累计击键等。
- 支持“从数据页直接开专练”，看到薄弱键位就能马上补。

### 3. 词库系统

项目内置 5 套可选词库，单词类和短语类训练都会跟随词库选择：

| 词库 | 定位 | 适合谁 |
| --- | --- | --- |
| 小学 | 常见生活词、课堂词、低门槛短语 | 起步用户、低年级训练 |
| 初中 | 校园、出行、日常表达高频词 | 有一定基础，想扩大常用词面 |
| 高中 | 学术性更强、表达更抽象 | 需要更高强度词汇训练 |
| CET-4 | 大学英语四级常见高频词 | 通用大学英语训练 |
| CET-6 | 偏进阶的大学英语词汇 | 更高强度单词输入训练 |

### 4. 成长与数据系统

- 键位热力图：直接看到 26 个字母哪些稳、哪些虚、哪些一碰就慌。
- 最近趋势图：查看近期训练表现变化，不靠“我感觉今天状态不错”来判断。
- 解锁进度展示：当前难度差多少能解锁，界面会明明白白告诉你。
- 本地持久化：训练数据保存到浏览器本地，关掉页面也不会当场失忆。
- 数据导出与重置：适合备份、迁移、演示或从零重新开始。

### 5. 可访问性与体验

- 音效开关：想热闹一点可以开，想安静练也没问题。
- 色盲友好模式：热力图和状态色彩更易区分。
- 模式卡与选项高亮：当前选择更明确，不会点着点着忘了自己选了啥。
- 街机提示优化：太空侵略者支持更清晰的节奏提示和 Boss 前过渡提醒。

## 玩法说明

### 基础练习

如果把整个项目比作一套训练计划，那基础练习就是热身、校准和纠偏区。

- 进入后先选择难度、时长和词库。
- 系统会给出字母、单词或短语提示。
- 你输入正确，界面给正反馈；输入错误，界面会继续指向正确键位。
- 虚拟键盘会高亮，指法提示会告诉你大致该用哪根手指。
- 适合起步、纠错、补某几个老是按错的键位。

### 水果忍者

这是给“手速”和“节奏感”准备的。

- 屏幕会出现带字母的水果。
- 按对键，水果被切开；按错或者打到炸弹，会吃惩罚。
- 连续命中会形成连击，得分更高，反馈也更带劲。
- 越高难度，节奏越快，留给反应的时间越少。

适合你觉得自己“认识键盘了，但手还没跟上脑子”的阶段。

### 太空侵略者

这是给“持续输出”和“单词稳定度”准备的。

- 屏幕上的敌机会分波次下降，不会一上来就像开会点名一样全体冲锋。
- 每个敌机携带一个单词，你需要连续输入，把它准确击落。
- 系统会锁定当前目标，避免你打一半又去追别的目标，最后谁也没打完。
- 在 Boss 前会有明显过渡提示，让节奏更可预期。
- 模式会跟随所选词库，适合做分层词汇训练。

适合你已经不满足于单个字母，希望把输入拉长成“稳定输出链路”的阶段。

### 数据中心

这是项目最不像游戏、但最值钱的部分。

- 看热力图，知道薄弱键位在哪。
- 看趋势图，知道最近是稳步进步，还是今天手有点飘。
- 看今日重点键位，知道下一轮该优先补什么。
- 直接从数据页启动强化练习，把“发现问题”变成“立刻处理问题”。

一句人话：不是练完就散会，而是练完以后知道下一轮该练啥。

## 难度与解锁

项目不是“所有难度一股脑全开”，而是按熟练度逐步解锁。

| 难度 | 状态 | 解锁逻辑 |
| --- | --- | --- |
| 启程 | 默认开放 | 直接可用，适合起步和建立键位地图 |
| 巡航 | 需解锁 | 当 50% 键位达到 L2 时解锁 |
| 王牌 | 需解锁 | 当 80% 键位达到 L4 时解锁 |

界面会显示当前距离下一档还差多少，不需要靠猜，也不需要靠玄学。

## 推荐使用流程

如果你想把这个项目用得更顺手，建议按下面这条路线来：

1. 先进入主菜单，选好难度、时长和词库。
2. 从“推荐训练”开始，完成今天第一轮。
3. 训练结束后进入数据中心，看热力图和今日重点键位。
4. 对准最弱键位补一轮基础练习。
5. 手感稳定后再去水果忍者或太空侵略者，把速度和持续输出拉起来。

这套流程的核心不是“多玩”，而是“玩完以后知道下一步干什么”。

## 使用说明

### 环境要求

- Node.js 18 及以上版本
- npm 9 及以上版本

### 本地启动

```bash
npm install
npm run dev
```

启动后打开终端里显示的本地地址即可。

### 生产构建

```bash
npm run build
```

### 一键完整校验

```bash
npm run verify
```

这条命令会依次执行 `lint`、`typecheck`、`test` 和 `build`，适合提交前做一次总检查。

## 愿景与目标

`Max Typer` 想做的，不只是一个“打字小游戏合集”。

它更像一个开源的训练底座，目标包括：

- 让键盘训练更有趣，但不牺牲训练价值。
- 让每一次错误都能进入数据闭环，而不是错完就算。
- 让成长可视化，让孩子看得懂、家长看得见、老师能复盘。
- 让这个项目本身也成为一个值得复用和继续扩展的开源教育游戏样板。
