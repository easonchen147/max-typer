# 《键盘冒险岛》儿童打字游戏策划书

## 说明
版本：1.1
目标用户：4-12岁儿童
核心理念：以数据为核心，动态巩固薄弱环节，通过可视化键盘让学习进度一目了然


## 1. 项目概述
《键盘冒险岛》精简版聚焦于三个核心练习模块和一个数据统计面板，通过游戏化交互激发儿童打字兴趣，同时利用数据驱动实现个性化学习路径。应用收集孩子每次按键的准确率、反应时间、错误频率等数据，生成个人学习画像，并动态调整后续练习内容，重点强化容易混淆或反应慢的字母和单词，帮助孩子高效提升打字能力。数据统计面板采用键盘热力图形式，直观展示每个字母的熟练程度，让孩子和家长对薄弱环节一目了然。

## 2. 核心功能模块
#### 模块一：基础练习模式（可视化键盘引导）
玩法：
屏幕上部分显示一个放大的字母或单词，下部分显示一个虚拟键盘（与真实键盘布局一致）。孩子根据提示按下对应键，虚拟键实时高亮反馈。此模式专为初学者设计，帮助建立字母与键位的视觉对应关系。

难度与时长选择：

难度：

初级：仅练习字母，随机出现大写/小写字母，可设置只练习特定字母组（如基准键位）。

中级：练习简单单词（如 cat、dog），需逐个字母输入，每正确一个字母该字母变绿。

高级：练习短句（如 “I am a boy”），强化连续输入。

时长：可选 3分钟、5分钟、10分钟或无限时。

交互反馈：

正确输入：字母跳动，虚拟键发出彩色涟漪，播放轻快“叮”声。

错误输入：虚拟键闪烁红色，同时正确键持续高亮，引导孩子找到正确位置；连续错误3次后，系统用语音提示（如“按左边的A键”）。

完成单词/短句后，播放胜利音效，屏幕撒花。

数据收集：

每个按键的正确/错误次数。

每个按键从显示到按下的反应时间。

单词输入完整度和耗时。

#### 模块二：水果忍者·打字版（字母反应训练）
玩法：
屏幕上不断蹦出带有字母的水果（苹果、香蕉等），孩子需快速敲击对应字母键“切开”水果。切中水果得1分，切中炸弹暂停2秒并扣1分。连击可获得额外分数。

难度与时长选择：

难度：

休闲：只出现 A~H 字母，水果下落慢。

普通：出现全部字母，速度适中。

挑战：速度加快，同时出现多个水果，需快速连续输入。

时长：可选 3分钟、5分钟、10分钟或无限时。

交互反馈：

正确：水果被切开，果汁飞溅，播放“咔嚓”声。

错误：水果抖动并发出“嘟”声。

连击：屏幕出现“COMBO x2”特效，得分加倍。

数据收集：

每个字母的正确/错误次数（切中水果/切中炸弹或错过）。

每个字母从出现到按键的反应时间。

连击次数、总分。

#### 模块三：太空侵略者·打字版（单词综合训练）
玩法：
屏幕上方排列外星飞船方阵，每艘飞船上显示一个单词。孩子需输入正确单词发射子弹击落飞船。全部飞船击落后进入下一关；飞船抵达底部则游戏结束。

难度与时长选择：

难度：

简单：单词库为CVC单词（cat、dog），飞船移动慢。

中等：单词库含CVCe和多音节单词，飞船速度中等。

困难：单词库含短句，飞船速度快，并有特殊飞船（护盾、加速等）。

时长：可选 3分钟、5分钟、10分钟或无限时（无限时模式下以通关为目标）。

交互反馈：

正确输入：飞船爆炸，播放“嘭”声，屏幕轻微震动。

错误输入：飞船闪烁红光，子弹未发射，宠物叹气。

数据收集：

每个单词的正确/错误次数。

每个单词的输入耗时。

每个字母在单词环境下的正确率（可分解到字母级别）。

关卡完成度、总分。

#### 模块四：数据统计面板（键盘热力图）
核心功能：
汇总孩子在所有模块中的练习数据，生成可视化的个人学习报告，核心是一张彩色键盘图，用不同颜色表示每个字母的熟练程度。同时提供其他统计图表，全面展示学习进度。数据统计面板不仅用于回顾，还驱动其他模块的个性化出题。

4.1 键盘热力图设计
键盘布局：显示标准美式键盘（包含26个字母键），每个键位根据其熟练程度填充不同颜色。

颜色映射：采用从红到绿的渐变，红色表示熟练度低（需加强），绿色表示熟练度高（已掌握），中间用黄色过渡。可自定义色阶。

熟练度指标：可选用正确率（如正确率<60%为红，60%-80%为黄，>80%为绿），也可综合正确率和反应时间。

默认使用正确率，因为直观易懂。后续可切换为反应时间视图。

动态更新：每次练习结束后，热力图实时更新，孩子可以立即看到进步。

交互功能：

点击某个字母键，可查看该字母的详细统计：正确次数、错误次数、平均反应时间、近7天趋势。

长按字母键可进入“强化练习”模式，直接跳转到基础练习模式并锁定该字母进行专项训练。

提供“显示最薄弱5个字母”按钮，自动高亮需要重点练习的键位。

4.2 其他统计维度
单词熟练度云图：用词云形式展示已学单词，单词大小表示熟练度，颜色表示正确率。

整体趋势图：

折线图：每日总打字量、平均正确率的变化。

柱状图：各模块练习时长占比。

进步勋章：当某个字母从红色变为绿色时，弹出祝贺动画，给予虚拟奖励（如星星），激励孩子持续进步。

4.3 数据反馈机制
模块一出题：优先选择热力图中红色和黄色的字母，以及包含这些字母的单词。可设置“强化模式”，专门练习最薄弱的5个字母。

模块二水果出现概率：水果上的字母概率与热力图颜色挂钩，红色字母出现频率更高，鼓励练习；绿色字母适当减少。

模块三单词库：优先选择熟练度低的单词（根据单词内字母的熟练度综合计算），并动态调整难度。例如，若单词“cat”中的c和t是红色，a是绿色，则“cat”的优先级提升。

### 3. 数据驱动个性化学习闭环
#### 3.1 数据采集
每个模块每次练习时，实时记录：

用户ID、时间戳、模块类型。

当前难度、时长设定。

每个按键的字母、是否正确、反应时间。

单词输入序列、完成情况。

#### 3.2 数据分析模型
字母维度：对每个字母统计正确率、平均反应时间、错误分布（按上下文、手指位置等）。

单词维度：按长度、音节、字母组成统计熟练度，并标记易错单词。

长期趋势：建立用户学习曲线，识别进步缓慢区域。

#### 3.3 动态出题策略
基础练习模式：

初始化：从全部字母随机抽取。

根据热力图数据，红色字母出现概率提升至3倍，黄色提升至1.5倍，绿色保持基础概率。

支持“只练薄弱字母”模式，连续10次正确后该字母移出强化列表。

水果忍者模式：

水果出现概率 = 基础概率 × (1 + 红色等级系数)。红色字母的水果更大、更鲜艳，增加视觉吸引力。

炸弹出现概率与整体正确率挂钩（正确率高则炸弹少，反之略多）。

太空侵略者模式：

单词库按照单词内字母的平均熟练度排序，优先抽取平均熟练度低于60%的单词，同时保证已掌握单词占20%以维持信心。

若连续正确3次，则从当前难度池中随机提升一个未掌握单词；若连续错误2次，则降低单词难度或给出提示。

#### 3.4 隐私与数据安全
所有数据存储在本地（IndexedDB），不上传云端。

家长可通过导出功能备份数据，或重置数据。

### 4. 技术实现要点
#### 4.1 技术选型
前端框架：React + TypeScript（用于界面和逻辑）。

游戏引擎：Phaser 3（用于水果忍者和太空侵略者模块，处理动画、碰撞、计时）。

虚拟键盘：HTML/CSS 绘制，通过JavaScript监听键盘事件并高亮。

数据存储：IndexedDB（存储用户档案、历史数据）。

图表库：ECharts 或 D3.js（用于数据统计面板的折线图、词云等），键盘热力图可用Canvas或SVG自定义绘制。

#### 4.2 模块间通信
数据层统一管理，通过 React Context 或 Redux 提供全局状态。

Phaser 游戏场景通过事件与React组件通信（如游戏结束触发数据保存）。

数据统计面板实时订阅数据更新，动态刷新热力图和图表。

#### 4.3 键盘热力图绘制
使用Canvas绘制键盘布局，每个键位是一个矩形区域，根据熟练度填充颜色。

监听Canvas的点击事件，获取点击坐标，判断哪个字母被点击，弹出详情。

或者使用SVG绘制，每个键位作为一个独立元素，方便添加交互。

#### 4.4 扩展性
后续可增加多孩子账户切换、家长报告导出、云端同步（可选）等功能。

如果需要打包为桌面应用，可使用 Tauri 包装，数据存储升级为 SQLite，但当前版本以Web应用为主，部署在本地或服务器。

### 5. 总结
本方案通过三个核心练习模块（基础引导、字母反应、单词综合）和一个数据统计面板（键盘热力图），构建了一个完整的数据驱动儿童打字训练系统。键盘热力图让孩子的学习进度和薄弱环节一目了然，既激励孩子努力变“红”为“绿”，也帮助家长和老师快速定位问题。动态出题策略确保每一次练习都针对孩子的最远发展区，最大化学习效率。游戏化包装保持了趣味性，数据闭环保证了教育深度，是一款兼具实用性和可玩性的儿童打字软件。
